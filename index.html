各授業回の学習事項

第1回 WWW と HTML

「事前準備の確認」、「Webページとは」、「HTMLについて」を学んだ。 事前準備の確認では、firefox、vscode、githubの設定について確認した。 WebページについてはURLの構成部品について学んだ。 HTMLについては、HTML要素、HTML文書の構成について学び、演習を行なった。

演習の成果：演習1-4: HTMLのさまざまな装飾

第2回 CSS による Web ページの装飾

CSSによって見た目を装飾する方法について学んだ。 CSSをHTMLに適用する方法は外部スタイルシート、内部スタイルシート、インラインスタイルの三つがあるが、 基本的に外部スタイルシートを使用する。 CSSはセレクタ、プロパティ名、プロパティ値によって構成される。

演習の成果 : 練習問題2-3

第3回 JavaScriptの基本

今回はJavaScriptについて簡単な演習を通して学んだ。 JavaとJavaScriptは共通する点もあるが、異なる言語である。 HTMLにJavaScriptを導入するには三つの方法があるが、 演習では主に外部でファイルを作成する方法を使う。 Firefox上のウェブ開発ツールからコンソールを開くことでJavaScriptの実行を確認することができる。 一度そこで実行してからVScodeに書き込むことが望ましい。
