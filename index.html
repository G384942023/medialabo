数当てゲーム

ランダムで生成される1~10の数を三回まで予想して答えと一致しているかを判定する。 第三回課題では予想の値を４に固定して回数と予想が正解しているかによって変化する文章をコンソール上で表示されるようプログラムした。 また、不正解の場合は答えを上回っているか下回っているかを判定してヒントを出し、正解の場合は終了するようにした。 第四回課題ではDOMを使用して数当てゲームの結果をコンソールではなくページに表示されるよう修正した。予想は４で固定され、判定の回数を一回に減らした。 第五回課題で予想した数値と入力するテキストボックスと回答するボタンを設置し、予想の値と答えを比較してヒントを出す数当てゲームを完成させた。

数当てゲームのページ: kazuate.html

各授業回の学習事項

第1回 WWW と HTML

第1回講義では「事前準備の確認」、「Webページとは」、「HTMLについて」を学んだ。 事前準備の確認では、firefox、vscode、githubの設定について確認した。 WebページについてはURLの構成部品について学んだ。 HTMLについては、HTML要素、HTML文書の構成について学び、演習を行なった。

演習の成果：演習1-4: HTMLのさまざまな装飾

第2回 CSS による Web ページの装飾

第2回講義ではCSSによって見た目を装飾する方法について学んだ。 CSSをHTMLに適用する方法は外部スタイルシート、内部スタイルシート、インラインスタイルの三つがあるが、 基本的に外部スタイルシートを使用する。 CSSはセレクタ、プロパティ名、プロパティ値によって構成される。

演習の成果 : 練習問題2-3

第3回 JavaScriptの基本

第3回講義ではJavaScriptについて簡単な演習を通して学んだ。 JavaとJavaScriptは共通する点もあるが、異なる言語である。 HTMLにJavaScriptを導入するには三つの方法があるが、 演習では主に外部でファイルを作成する方法を使う。 Firefox上のウェブ開発ツールからコンソールを開くことでJavaScriptの実行を確認することができる。 一度そこで実行してからVScodeに書き込むことが望ましい。

演習の成果 : 練習問題3-6

第4回 DOM と JavaScript

第4回講義ではDOMについて学んだ。主に演習を中心に進め、プロジェクト課題、数当てゲームの改良を行なった。 DOMによって、要素の検索、テキストの変更、CSSプロパティの変更、二つの方法での新しい要素の追加、要素の削除を行うことができる。

演習の成果 : 練習問題4-2から4-5

第5回 イベント処理

第5回講義ではイベントが発生したときにプログラムを動作させる処理の方法について学んだ。 イベントとはボタンをクリックする、マウスポインタが重なる、キーを押すなどの操作のことである。 それによってテキストボックスやボタンを設置することができる。

演習の成果 : 練習問題5-3
